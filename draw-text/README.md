# 向图片中插入指定文字

用法:

```
usage: insert.py [-h] -i INPUT -B BACKGROUND -o OUTPUT -f FONT [-x X] [-y Y] [-r RED] [-g GREEN] [-b BLUE] [-s SIZE]

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Text list filepath
  -B BACKGROUND, --background BACKGROUND
                        Background image path
  -o OUTPUT, --output OUTPUT
                        Output path
  -f FONT, --font FONT  font path
  -x X, --x X           Location x
  -y Y, --y Y           Location y
  -r RED, --red RED     Red
  -g GREEN, --green GREEN
                        Green
  -b BLUE, --blue BLUE  Blue
  -s SIZE, --size SIZE  Size
```
